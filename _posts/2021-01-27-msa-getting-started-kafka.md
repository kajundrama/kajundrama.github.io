# Github Pages 만들기
일반적으로 github의 Page를 만들기 위해서는 Repository명을 username.github.io로 만들면 
자동적으로 Pages 가 생성됩니다.

Enterprise Github에서도 동일하게
[username].[orgname] 로 작성을 하면 됩니다.
성성 후 Page에 접근하는 URl은 
https://[orgname].pages/[username]


```text
README.md 과 같은 파일을 반드시 올려야 합니다.
```
